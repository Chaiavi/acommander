| Key         | Action                                                  |
|-------------|---------------------------------------------------------|
| **F2**      | Rename the selected file                                |
| **F3**      | View the selected file                                  |
| **F4**      | Edit the selected file                                  |
| **F5**      | Copy the selected file to the other panel               |
| **F6**      | Move the selected file to the other panel's path        |
| **F7**      | Create a new folder                                     |
| **F8 / Del**| Delete the selected file or folder                      |
| **F9**      | Open PowerShell or Command Prompt                       |
| **F10**     | Exit the application                                    |
| **F11**     | Pack the selected file as a `.zip` into the other panel |
| **F12**     | Unpack the selected file to the other panel             |
| **ALT+F11** | Split a large file into `.7z` parts                     |
